The `.pop_back()` function is a method in C++'s `std::string` class that **removes the last character** from the string.

| Syntax                    | Description                                                                                                          |
| ------------------------- | -------------------------------------------------------------------------------------------------------------------- |
| `str.erase(pos, len);`    | Removes `len` characters starting from index `pos` (0-based).                                                        |
| `str.erase(position);`    | Removes the single character at the given iterator `position`.                                                       |
| `str.erase(first, last);` | Removes characters in the range `[first, last)`, where `last` is one position past the last character to be removed. |
You need to include the `<string>` and `<iostream>` headers to use it:
```
#include <iostream>
#include <string>
```
